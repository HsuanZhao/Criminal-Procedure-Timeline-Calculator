# 中国刑案流程节点计算器 (Criminal Procedure Timeline Calculator)

> 一个面向刑事案件家属、当事人及法律从业者的轻量级 Web 工具，基于《中华人民共和国刑事诉讼法》法定上限期限，自动推算刑事拘留、侦查、审查起诉、一审、二审等各阶段的关键节点日期。

## 📌 项目简介

刑事案件中，家属最焦虑的问题往往是：“人什么时候能出来？”“为什么这么久还没消息？”“下一步要等到什么时候？”  
然而，刑事诉讼流程涉及大量法定期限，且不同情形（如流窜作案、多次作案、结伙作案）会导致期限大幅延长，家属很难自行准确推算。

**本工具的核心功能**：  
输入《拘留通知书》上的拘留日期，选择是否属于“流窜/多次/结伙作案”情形，即可一键生成一份按 **法定期限上限（最长）** 推演的完整时间线，涵盖：

- 提请批准逮捕的最迟日期（3日 / 30日 + 检察院7日）；
- 逮捕后一般侦查羁押期限及延长节点（2+1+2+2 个月）；
- 审查起诉期限（1个月 + 15日，含退回补充侦查）；
- 一审审理期限（2+1+1 个月）；
- 上诉期（10日）及二审审理期限（2+2 个月）。

同时，每个节点都附有 **“家属可以做”** 和 **“律师可以做”** 的实务建议，并内嵌法条原文（点击即可查看），帮助用户理解每个期限的法律依据。

> ⚠️ **重要声明**：本工具仅为“理想状态”下的期限推算，实际办案中因退回补充侦查、发现新罪行、管辖权变更、重新计算羁押期限等复杂情况，实际日期可能显著早于或晚于计算结果。**本工具仅供参考，不能替代专业法律意见**，请务必以办案机关的实际通知为准。

## ✨ 功能特点

- **一键推算**：选择拘留日期和情形类型，自动生成完整时间线。
- **两种情形覆盖**：
  - **普通情形**（单人、单次、本地作案）：提请批捕最长 3+7 日；
  - **重大嫌疑情形**（流窜/多次/结伙作案）：提请批捕最长 30+7 日（俗称“黄金37天”）。
- **可视化节点展示**：每个阶段以卡片形式清晰显示日期、法律依据和实务指引。
- **法律依据内嵌**：点击“查看法条依据”即可弹出《刑事诉讼法》对应条款原文，无需额外查询。
- **响应式设计**：适配手机、平板、电脑，方便家属在移动端随时查看。
- **纯前端应用**：无需后端服务器，不收集任何用户数据，隐私安全。

## 🚀 使用方法

### 在线使用（推荐）
直接访问部署后的 HTML 页面即可使用（无需安装任何软件）。

### 本地运行
1. 下载 `刑案流程节点计算器.html` 文件；
2. 双击用浏览器（Chrome / Edge / Safari 等）打开；
3. 按页面指引输入日期、选择情形，点击“开始计算”。

## ⚖️ 计算逻辑说明

本工具严格依据《中华人民共和国刑事诉讼法》及相关司法解释的 **最长期限上限** 进行推演，默认采用“办案机关充分行使法定延长期限”的保守估算，旨在给家属一个“最晚可能到哪一天”的心理预期。

| 阶段 | 依据条款 | 期限设定 |
|------|----------|----------|
| 拘留后提请批捕 | 刑诉法第91条 | 普通：3日（可延至7日）；重大：30日 |
| 检察院批捕决定 | 刑诉法第91条 | 7日 |
| 侦查羁押一般期限 | 刑诉法第156条 | 2个月 |
| 案情复杂延长 | 刑诉法第156条 | +1个月（上一级检察院批准） |
| 特别重大复杂延长 | 刑诉法第158条 | +2个月（省级检察院批准） |
| 十年以上可能再延长 | 刑诉法第159条 | +2个月（省级检察院批准） |
| 审查起诉一般期限 | 刑诉法第172条 | 1个月 |
| 重大复杂延长 | 刑诉法第172条 | +15日 |
| 退回补充侦查（每次） | 刑诉法第175条 | 1个月（重新计算审查起诉期限） |
| 一审一般审限 | 刑诉法第208条 | 2个月 |
| 一审复杂延长 | 刑诉法第208条 | +1个月（上一级法院批准） |
| 一审特殊再延长 | 刑诉法第208条 | +1个月（最高法批准） |
| 上诉/抗诉期 | 刑诉法第211条 | 10日（自收到判决书次日起） |
| 二审一般审限 | 刑诉法第232条 | 2个月 |
| 二审延长 | 刑诉法第232条 | +2个月（省级高院批准） |

> 注意：本工具以 **侦查终结后立即移送审查起诉**、**审查起诉期满后立即提起公诉** 等理想衔接为假设，实际中可能存在间隔，因此所有后续日期均为理论最早或最晚参考。
